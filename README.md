# k-data
JS常用数据结构实现。

## Install
```
yarn add @depeng9527/k-data
```

## Stack

| name    | type | description        | return  |
| ------- | ---- | ------------------ | ------- |
| size    | 属性 | 栈的大小长度       | number  |
| isEmpty | 属性 | 栈是否为空         | boolean |
| push    | 方法 | 添加               | void    |
| pop     | 方法 | 移出栈第一个元素   | T       |
| clear   | 方法 | 清除栈中所有       | void    |
| peek    | 方法 | 返回栈中第一个元素 | T       |
